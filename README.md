#Quack Attack *words first duck powered posture checker Featuring the posture police

A slouch detector using HTML, JavaScript, and CSS and a real time pose detector using TensorFlow.js MoveNet. It learns what your good posture looks like and when you slouch it triggers a funny feature...

FEATURES

POSE DETECTOR: using tensor flow and MoveNet lightning
posture calibration: takes a "pic" of your perfect posure calculating your nose-shoulder vertical distance and stuff
Ducks: Ducks: ducks:


Tech stack
Front-end HTML5, CSS3, JavaScript
ML library: TensorFlow
Pose detector model: MoveNet

How to run it locally
1. clone the repository using:
git clone https://github.com/YOUR_GITHUB_USERNAME/duck-posture-police.git
2. open the project using VS code (i prefer it)
3. Install the live server extension
4. Right clikc the index.html and press open with live server.
5. Grant camera permissions sit up straight and press the calibrate button.
